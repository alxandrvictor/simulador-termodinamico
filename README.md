# simulador-termodinamico
Simulador termodinâmico em Fortran para ciclos de refrigeração (R134a/R404A). Implementação de Newton-Raphson e Eliminação de Gauss aplicada à simulação da compressão, dimensionamento de hardware e análise estocástica (Monte Carlo). Trabalho Final do curso de Métodos Numéricos do Prof. Dr. Rafael Gabler Gontijo (Ciência e Brisa).
